# SECURITY SPEC: Correção de Visibilidade do Texto nos Inputs de Login e Senha

- Sem impacto ou vulnerabilidade de segurança. O campo de senha continuará ocultando caracteres via `type="password"`, e a visibilidade visual referida se aplica aos caracteres mascarados padrão do navegador (ex: bolinhas/asteriscos) e ao texto digitado no campo de email.
