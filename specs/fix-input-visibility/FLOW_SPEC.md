# FLOW SPEC: Correção de Visibilidade do Texto nos Inputs de Login e Senha

## Fluxo do Sistema
1. Usuário acessa `http://localhost:3000/login`.
2. Usuário foca no campo de input "E-mail" ou "Senha".
3. Usuário digita os dados.
4. O texto deve aparecer legível e contrastante em tempo real no campo de entrada de dados.
