# PRODUCT SPEC: Nova Paleta de Cores do Projeto

## Objetivos da Feature
- Definir e registrar a nova paleta de cores corporativa/marca para o projeto, garantindo consistência visual e identidade harmônica entre todas as páginas e componentes.
- Garantir alto contraste e flexibilidade na aplicação de elementos de destaque, planos de fundo e textos.

## Cores Escolhidas
1. `#a6988a` (Sand / Neutro Terroso)
2. `#88a19f` (Sage / Cinza Sage)
3. `#6aabb5` (Teal / Azul Teal)
4. `#4bb4ca` (Cyan / Ciano Médio)
5. `#1ec3ea` (Vibrant / Ciano Vibrante)

## Requisitos Funcionais
- A paleta deve estar disponível para uso em todos os componentes e páginas do frontend.

## Regras de Negócio e Critérios de Aceite
- Os componentes de UI devem ser capazes de referenciar estas cores nativamente via classes do Tailwind CSS e variáveis CSS nativas.
